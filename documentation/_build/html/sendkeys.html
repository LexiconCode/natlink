sendkeys¶

The sendkeys function sends keystrokes to the foreground window.

The format that is used in Unimacro and Vocola is translated into Dragonfly format, and the dragonfly.actions.action_key.Key class performs the actions.

This replaces the natlink.playString function of Natlink and the SendInput of Vocola.

sendkeys module¶

Sending keystrokes to the foreground window.

Usage: from natlink.sendkeys import sendkeys

and then: sendkeys"("keystrokes")

This module now adopts the Dragonfly action_key module, so “{alt+w}” is (in the function) converted to “a-w” etc.

(Quintijn Hoogenboom, 2021-04-04)

sendkeys(keys)[source]¶

sends keystrokes via dragonfly class action_key.Keys

Keystrokes following Unimacro/Vocola convention are translated into Dragonfly notation:

"{shift+right 4}" is translated to "s-right:4".

Multiple keystrokes, pauses before the individual keys and a pause after all the keys can be given in Dragonfly notation:

"{down/25:12/100}" presses the down key 12 times, with a pause before each key of 25/100 seconds and a pause of 1 second (100/100 seconds) after the 12 down presses

Extra notation (Quintijn):

"!" at the end of a chord (the keystrokes definition between the braced, { and }, triggers the use_hardware=True event, comparable with the SendSystemKeys mechanism of Dragon.
